Release Notes¶

Branch Volume 0.0.2¶

October 14, 2015

Event Identification¶

The Branch Volume has been improved to identify normal, payroll, and holiday event types. Within our database system, we have been able to match the days and weeks associated to a transaction date. This means you now have the ability to identify and compare event types to the total transaction average for a given time period.

Caveats with Event Dates¶

The Branch Volume Dashoboard focuses on factual data with as minimal assumptions as possible. With that, the current output result of an event type is set to reflect an event on a particular day. If there are no transactions on a given day, in the case of many holiday events, there will be no data provided.
